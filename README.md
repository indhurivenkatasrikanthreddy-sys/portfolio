# Portfolio Website

A personal portfolio website built with HTML, CSS, and JavaScript.

## Overview

This project is a responsive portfolio site with:

- A fixed navigation bar
- A hero section
- About, Skills, Projects, Resume, and Contact sections
- Mobile navigation menu
- Simple animations and interactive effects

## Project Files

- `index.html` - main structure of the portfolio
- `styles.css` - custom styles and layout rules
- `script.js` - menu behavior, animations, form handling, and resume actions
- `resume.pdf` - downloadable/viewable resume file

## How to Run

1. Open `index.html` in your browser.

You can also use the VS Code Live Server extension for a better development experience.

## Customization

Update these items to make the portfolio fully yours:

- Name, tagline, and intro text in `index.html`
- Skills, projects, and contact details in `index.html`
- Images such as `profile.jpeg` and project images
- Resume file by replacing `resume.pdf`

## Notes

- The resume buttons are connected to `resume.pdf`
- Tailwind is loaded through CDN in `index.html`
- Custom design styling is handled in `styles.css`
